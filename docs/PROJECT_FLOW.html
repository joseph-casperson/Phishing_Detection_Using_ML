Database Decision — PostgreSQL vs MongoDB

PostgreSQL — Primary Store

Structured tabular data with a fixed schema maps perfectly to a relational DB. Use this for your main URL and features tables. Enables the SQL queries required by Ch. 6, JOINs, aggregations, and Python's psycopg2/sqlalchemy.

✓ RECOMMENDED for primary data

MongoDB — Supplementary Store

Store your model run results and evaluation metrics as JSON documents. Each model run becomes a document: hyperparameters, split used, accuracy, F1, confusion matrix. This gives you a natural "experiment tracker" with no fixed schema.

✓ RECOMMENDED for experiment logs
Why both? Using PostgreSQL for structured feature data and MongoDB for experiment results is a legitimate, production-style architecture pattern. It demonstrates you understand when to use relational vs. document stores. It also satisfies Chapter 7 (NoSQL).

python — 02_preprocessing.ipynb (key snippets)
import pandas as pd
from sqlalchemy import create_engine

engine = create_engine("postgresql://phishuser:yourpassword@localhost/phishdb")
df = pd.read_sql("SELECT * FROM features JOIN urls USING (url_id)", engine)

# ── AUDIT BEFORE ──────────────────────────────────────
print(f"Shape: {df.shape}")
print(f"Duplicates: {df.duplicated().sum()}")
print(df.isnull().sum())

# ── CLEAN ─────────────────────────────────────────────
before = len(df)
df = df.drop_duplicates()
df['url_string'] = df['url_string'].str.lower().str.strip()
df = df.dropna(subset=['url_length', 'label'])
after = len(df)
print(f"Removed {before - after} rows")

# ── EXPORT ────────────────────────────────────────────
df.to_csv("data/clean/clean_urls.csv", index=False)
df.to_sql("clean_features", engine, if_exists="replace", index=False)

python — modeling loop with MongoDB logging
from sklearn.ensemble import RandomForestClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.svm import SVC
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report
from pymongo import MongoClient
import datetime

client = MongoClient("mongodb://localhost:27017/")
runs = client.phishdb.model_runs

models = {
    "LogisticRegression": LogisticRegression(max_iter=1000),
    "RandomForest":      RandomForestClassifier(n_estimators=100),
    "SVM":               SVC(kernel='rbf'),
}
splits = [0.70, 0.80, 0.90]
